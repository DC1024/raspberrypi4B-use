首先我们需要使用以下命令，进入树莓派设置

```text
sudo raspi-config
```

找到「Interfacing Options」，找到「VNC」并启用 VNC 。

然后我们访问 [VNC Viewer](https://link.zhihu.com/?target=https%3A//www.realvnc.com/en/connect/download/viewer/) 的下载界面，下载对应的操作系统版本软件

比如我要用 Windows 系统连接树莓派，就下载 Windows 版本的 VNC Viewer

然后在 VNC 里输入树莓派的 IP 的地址，并输入对应用户名密码进行登录
