打开 Wallabag ，点击左侧的「API clients management」，点击 「CREATE A NEW CLIENT」。「Name of the client」随意填写，比如我要给我的 Edge 浏览器创建这个 API ，就可以填写”Wallabag API for Edge“
![](http://image.dcchen.top/www/wwwroot/easyImages/image2021/09/11/pqkehg.png)
然后我们就得到了「Client ID」和「Client Secret」
![](http://image.dcchen.top/www/wwwroot/easyImages/image2021/09/11/ps7ufm.png)

然后在浏览器上搜索拓展：「Wallabag」并安装
![](http://image.dcchen.top/www/wwwroot/easyImages/image2021/09/11/pshix7.png)
点击安装好的拓展，输入对应内容
![](http://image.dcchen.top/www/wwwroot/easyImages/image2021/09/11/qjmn1m.png)
点击一下获取 token，尝试打开其它网页，然后单击一下拓展按钮，就能看到 Wallabag 正在工作
![](http://image.dcchen.top/www/wwwroot/easyImages/image2021/09/11/ql4fav.png)
