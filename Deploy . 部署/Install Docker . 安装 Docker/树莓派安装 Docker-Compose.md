更新 python

[[如何升级树莓派 Python]]

安装 pip

```text
sudo apt-get install python-pip -y
```

安装 libffi-dev

```text
sudo apt-get install libffi-dev
```

安装 docker-compose

```text
sudo pip install docker-compose
```

查看 docker-compose 版本

```text
docker-compose version
```
